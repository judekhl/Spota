Spota is a mobile application for real-time parking availability in Haifa, Israel.



The app helps users:

\- Find nearby parking lots

\- See available parking spaces

\- See prices

\- See open/closed status

\- Navigate to a selected parking lot using Waze or Google Maps



The operator side allows parking-lot operators to:

\- Update available spaces manually

\- Update price

\- Update open/closed status



The first version will use manual updates, not real parking-gate integrations.

The goal is to build a working demo for Haifa parking lots.
